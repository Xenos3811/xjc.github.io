Readding

Readding 是一个类似豆瓣图书的综合图书推荐平台

这是一个学习项目,是针对于golang与vue的相关技术知识的巩固

选型

后端

  • Golang: 简单,快,代码优雅!
  • Fiber: 一个web服务框架,相比于gin有更多的功能与插件

前端

  • TypeScript
  • Vue3 +
  • ElementPlus +
  • Pinia
  • Vite

其他

在选型过程中发现的一些其他组件,并没有运用,只做记录

  • Tailwindcss: 一个Css3框架
  • Quasar: 一个国外的UI组件框架.有桌面化UI库
  • NuxtUi: 一个国外的UI库,很漂亮,不过需要付费
  • Arco: 字节的UI框架
  • TDSIGN: 腾讯的UI框架
  • Naive UI: 一个Vue3组件库
  • PrimeVue: UI
  • shadcn

UI框架的选择实在太多了,国内的多而杂且没什么特色都差不太多,NaiveUI标榜最快但是个人开发.
对比起来Element算是成熟,开源的最早有经验沉淀.使用的人也最多

初始化项目

初始目录

Server

  1. 创建一个main.go文件,和一个go.mod文件
  2. main.go文件指定包为package main ! package 一定要为main,才可以执行.
  3. go.mod 文件指定包为server. package server
  4. 安装fiber框架,在server目录下 终端运行 go get github.com/gofiber/fiber/v2

简单尝试

简单尝试主要试一试vue,element,fiber的基础能力

web

vue 项目的程序主入口是index.html,它里面有一个id=app的div,它引入了main.ts文件

main.ts中创建了一个vue的单页面实例
为这个实例添加路由
app.user(router)
为这个实例添加状态管理
app.use(createPinia)
为这个实例添加elementUI组件
app.use(Element)
将vue实例挂载在上面的div标签中
app.mount(‘#app’)

Server

在上面我们创建了一个基于fiber的web服务,启动服务后,使用curl工具请求可以接口能成功响应

在开始之前,有一个问题,现在的server服务没有热加载的功能,即每次修改代码后需要重新启动,这样比较麻烦.

我们先为之加上一个热加载的功能,使用air工具

现在在main.go的main方法中添加一个POST接口
